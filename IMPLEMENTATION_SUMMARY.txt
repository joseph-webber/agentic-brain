## SECURITY MODEL

┌─────────────────────────────────────────┐
│ Chatbot Role (ADMIN/USER/DEVELOPER)     │ ← First layer
├─────────────────────────────────────────┤
│ API Access Controller                    │ ← Second layer (rate limiting, scopes)
├─────────────────────────────────────────┤
│ External API (WordPress/WooCommerce)     │ ← Third layer (role permissions)
├─────────────────────────────────────────┤
│ Database/File System                     │ ← Protected resource
└─────────────────────────────────────────┘

## ROLE BREAKDOWN

ADMIN:
  ✅ Full machine access (shell, files, YOLO)
  ✅ Can access any API
  ✅ Can configure integrations
  ✅ Use case: Joseph (system administration)

USER (CUSTOMER):
  ❌ NO machine access
  ✅ ONLY API access (WordPress, WooCommerce, etc.)
  ✅ Permissions controlled by API's role system
  ✅ Rate limited
  ✅ All access logged
  ✅ Use case: Customer support chatbots, store assistants

DEVELOPER:
  ✅ Limited machine access (safe directories only)
  ✅ Can access APIs for development/testing
  ✅ Dangerous commands blocked
  ✅ Use case: Power users, developers building with the brain

GUEST:
  ❌ NO machine access
  ✅ Read-only API access (public endpoints only)
  ✅ Very rate limited
  ✅ Use case: Anonymous website visitors, public help desk

## EXAMPLE USAGE

WordPress Content Bot:
```python
wp_client = create_wordpress_client(
    site_url="https://example.com",
    username="chatbot",
    app_password="xxxx xxxx xxxx xxxx",
    wp_role=WordPressRole.AUTHOR,
    chatbot_role=SecurityRole.USER,  # API-ONLY
)

# Can create and publish posts
await wp_client.create_post(title="Help Article", content="...")

# Cannot access file system ❌
# Cannot run shell commands ❌
```

WooCommerce Customer Assistant:
```python
wc_client = create_woocommerce_client(
    site_url="https://shop.example.com",
    consumer_key="ck_xxxxx",
    consumer_secret="cs_xxxxx",
    wc_role=WooCommerceRole.CUSTOMER,
    chatbot_role=SecurityRole.USER,
    customer_id=42,  # CRITICAL: Can only see THIS customer's orders
)

# Can view own orders
orders = await wc_client.list_orders()  # Only customer #42's orders

# Cannot view other customers' orders ❌
await wc_client.list_orders(customer=99)  # SecurityViolation!
```
