=============================================================================
                        COMMANDS IMPLEMENTED
=============================================================================

1. agentic-brain query "question" [OPTIONS]
   - Execute semantic search queries
   - Options: --top-k, --filters, --json
   - Example: agentic-brain query "What is ML?" --json

2. agentic-brain index <path> [OPTIONS]
   - Index documents for RAG system
   - Options: --recursive, --chunk-size, --overlap, --json
   - Example: agentic-brain index ./docs --recursive --json

3. agentic-brain eval <results-file> [OPTIONS]
   - Evaluate RAG results against metrics
   - Options: --json
   - Example: agentic-brain eval results.json --json

4. agentic-brain health [OPTIONS]
   - Check system health and component status
   - Options: --json
   - Example: agentic-brain health --json

5. agentic-brain config [OPTIONS]
   - Show/set configuration
   - Options: --get, --set, --json
   - Example: agentic-brain config --set chunk_size=1024

                        USAGE EXAMPLES
=============================================================================

Basic Query:
$ agentic-brain query "What is artificial intelligence?"

Query with Options:
$ agentic-brain query "machine learning" --top-k 10 --json

Index Documents:
$ agentic-brain index ./documents --recursive

Index with Custom Settings:
$ agentic-brain index ./docs --chunk-size 1024 --overlap 100

Evaluate Results:
$ agentic-brain eval results.json --json

Check System Health:
$ agentic-brain health

Health Check as JSON:
$ agentic-brain health --json

Show Configuration:
$ agentic-brain config

Get Specific Config:
$ agentic-brain config --get chunk_size

Set Configuration:
$ agentic-brain config --set top_k=10
