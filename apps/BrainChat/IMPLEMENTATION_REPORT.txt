  Cmd+1 → Jump to conversation area
  Cmd+2 → Jump to message input
  Cmd+3 → Jump to controls

  Cmd+M → Toggle microphone
  Cmd+L → Toggle microphone (alternative)
  Cmd+. → Stop speaking

MESSAGES:
  Return       → Send message

  Escape       → Clear input

  Cmd+↓        → Next message

  Cmd+N        → New conversation

  Cmd+?        → Show keyboard help
  Cmd+/        → Show keyboard help
