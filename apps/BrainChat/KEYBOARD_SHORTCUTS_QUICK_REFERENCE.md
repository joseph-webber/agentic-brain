# BrainChat Keyboard Shortcuts - Quick Reference

## Navigation
| Shortcut | Action |
|----------|--------|
| **Cmd+1** | Focus conversation area |
| **Cmd+2** | Focus message input |
| **Cmd+3** | Focus controls |

## Message Handling
| Shortcut | Action |
|----------|--------|
| **Cmd+Enter** | Send message |
| **Cmd+Shift+V** | Paste and send |
| **Cmd+Up** | Previous message in history |
| **Cmd+Down** | Next message in history |
| **Escape** | Clear input text |

## Conversation
| Shortcut | Action |
|----------|--------|
| **Cmd+N** | New conversation |
| **Cmd+K** | Clear conversation (confirmation required) |
| **Cmd+R** | Repeat last response |

## Audio
| Shortcut | Action |
|----------|--------|
| **Cmd+L** | Toggle microphone |
| **Cmd+M** | Toggle microphone (alternative) |
| **Cmd+.** | Stop speaking |

## Settings & Help
| Shortcut | Action |
|----------|--------|
| **Cmd+,** | Open settings |
| **Cmd+?** | Show keyboard shortcuts help |
| **Cmd+/** | Show help (alternative) |

## Tips & Tricks

### Quick Start
1. Press **Cmd+?** to see all shortcuts
2. Type a message and press **Cmd+Up** to recall it
3. Use **Cmd+1/2/3** to navigate without tabbing

### Message History Like Terminal
- **Cmd+Up Arrow** → Navigate back through messages
- **Cmd+Down Arrow** → Navigate forward
- **Escape** → Clear and start fresh
- Works just like `history` in terminal!

### Faster Workflow
- **Cmd+Shift+V** instead of `Cmd+V` then `Cmd+Enter`
- **Cmd+1** to read chat, **Cmd+2** to respond
- **Cmd+N** to start fresh (no manual clearing needed)

### VoiceOver Users
- Press **Cmd+1/2/3** for quick focus jumps
- Press **Cmd+?** to hear all shortcuts read aloud
- All actions announced with clear feedback
- Use **VO+U** (Web Rotor) for section navigation

### Power User Features
- **Cmd+R** - Re-read last response without re-sending
- **Cmd+M** - Toggle mic without leaving keyboard
- **Cmd+.** - Emergency stop (stops any speaking)
- **Cmd+N** - Start new conversation instantly

## Common Workflows

### Reviewing a Message
1. Send message with **Cmd+Enter**
2. Navigate back with **Cmd+Up** to edit
3. Type new version and **Cmd+Enter**

### Multi-Part Conversation
1. **Cmd+1** to read full chat
2. **Cmd+2** to respond
3. **Cmd+Up** to reference previous message
4. **Cmd+Enter** to send response

### Quick Copy & Respond
1. Copy text from another app
2. Switch to BrainChat
3. Press **Cmd+Shift+V** (paste + send)
4. Read response with **Cmd+1**

### VoiceOver Session
1. Enable VoiceOver (Cmd+F5)
2. Press **Cmd+?** to hear shortcuts
3. Press **Cmd+2** to focus input
4. Type and press **Cmd+Enter**
5. Press **Cmd+1** to hear response
