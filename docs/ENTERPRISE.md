## Deployment Options

### Self-Hosted (Recommended for Compliance)
```bash
pip install agentic-brain
# Your data never leaves your infrastructure
```

### Docker
```bash
docker pull agenticbrain/agentic-brain:latest
docker run -p 8000:8000 agenticbrain/agentic-brain
```

### Kubernetes
Helm charts available for production deployment with:
- Horizontal pod autoscaling
- Redis session storage
- Neo4j for knowledge graphs
- Prometheus metrics
