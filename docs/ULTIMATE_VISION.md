## ⚙️ SETUP PROFILES

**One command, perfect config:**

```bash
# Choose your profile
agentic-brain init --profile=secure      # Maximum security
agentic-brain init --profile=free        # Zero cost (Ollama only)
agentic-brain init --profile=performance # Maximum speed (GPU)
agentic-brain init --profile=cost        # Best $/token ratio
agentic-brain init --profile=enterprise  # Full compliance
agentic-brain init --profile=airlock     # Zero internet
```
