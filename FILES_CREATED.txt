INSTALLATION & USAGE
====================

All files are production-ready and require no additional installation.

To use:
    from agentic_brain.voice import get_user_region_storage, regionalize_text
    
    # Set region
    set_user_region("Adelaide", "South Australia")
    
    # Regionalize text
    text = regionalize_text("That is great!")

To run tests:
    pytest tests/test_user_regions.py -v

To run demo:
    python3 demo_user_regions.py

DATA STRUCTURE
==============

Storage location: ~/.agentic-brain/regions/

Files created on first use:
- user_region.json: Main configuration (custom + learned expressions, metadata)
- corrections.jsonl: Audit trail of user corrections (append-only)
- learnings.jsonl: Learning history with timestamps and confidence (append-only)

Total typical size: 5-50KB per user
